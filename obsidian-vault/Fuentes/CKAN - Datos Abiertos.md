# CKAN - Datos Abiertos

Portales de datos abiertos que proveen datasets municipales.

## Portales
- **CKAN PBA**: Portal de Datos Abiertos de la Provincia de Buenos Aires
- **CKAN Nacional**: Portal Nacional de Datos Abiertos (datos.gob.ar)

## Tipo de Datos
- Presupuestos municipales
- Indicadores socioeconomicos
- Datos de salud publica
- Infraestructura y obras
